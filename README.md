## Dependencies

- Node.js
- Express
- EJS
- bcrypt (2.0.0)
- body-parser
- cookie-session
- morgan

## Getting Started

- Install all dependencies (using the `npm isntall` command).
- Run the development web server using the `node express_server.js` command.
